# Quoting App – QA Automation Project

This project is a real-world SaaS quoting application originally built for business use and now used as a testing environment for QA automation.

## 🔧 Tech Stack
- Python
- Pytest
- Playwright
- REST APIs
- JSON

## 🧪 Test Automation Coverage

### UI Testing (Playwright)
- Login authentication
- Form submission
- Quote creation and editing
- Data validation

### API Testing (Pytest + Requests)
- Create quote endpoint
- Retrieve quotes
- Error handling and validation
- Backend data integrity

## 🧱 Test Architecture
- Page Object Model (POM)
- Pytest fixtures for setup/teardown
- Modular and reusable test structure

## 🚀 Live Application
https://quoting-app-zeta.vercel.app

## 🎯 Purpose
This project demonstrates:

- UI + API validation
- Real-world workflow testing
